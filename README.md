# CIS3190_A4

## Compiling and Running
* Python: 
  * To compile and run: `python3 -Wall calce.py`
* C: 
  * To compile: `gcc -std=c99 -lm -Wall calce.c`
  * To run: `./a.out`
* Fortran:
  * To compile: `gfortran -Wall calce.f95`
  * To run: `./a.out`
* Ada:
  * To compile: `gnatmake -Wall calce.adb`
  * To run: `./calce`
